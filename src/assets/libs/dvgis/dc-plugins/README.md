> http://dc.dvgis.cn

## Installation

`CDN`

```html
<!--Plugins Package-->
<script src="libs/dc-sdk/plugins/dc.plugins.min.js"></script>
```

`NPM / YARN`

```shell
yarn add @dvgis/dc-plugins
npm install  @dvgis/dc-plugins
```

```js
import DcPlugins from  'dvgis/dc-plugins/dist/dc.plugins.min' //Plugins Package
```

## Start

```js
DC.use(DcPlugins)
```
